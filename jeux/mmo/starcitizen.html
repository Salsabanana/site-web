Star Citizen

Star Citizen
Editeur : CIG
Sortie : non-fini
Site officiel

Star Citizen est un MMO futuriste en cours de développement. Il est developpé par Cloud Imperium Games. Ce jeu a pour but de créer une simulation la plus fidèle possible de ce que pourrait être la vie dans le futur.

Le projet a été lancé en 2012 par Chris Roberts et n'est toujours pas achevé.

Les joueurs évoluent dans notre univers 930 ans dans le futur, le jeu est loin d'etre fini, les joueurs peuvent pour le moment acceder à un seul systeme stellaire sur les dizaines prévus pour la realease mais cela reste immense et malgrés les bugs, les joueurs peuvent faire du combat spatial, des livraisons, du commerce, du minage...
