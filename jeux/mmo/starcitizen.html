Star Citizen

Star Citizen
Editeur : CIG
Sortie : non-fini
Site officiel

Star Citizen est un MMO futuriste en cours de développement. Il est developpé par Cloud Imperium Games. Ce jeu a pour but de créer une simulation la plus fidèle possible de ce que pourrait être la vie dans l'espace.

Le projet a été lancé en 2012 par Chris Roberts et n'est toujours pas achevé. Il est pour l'instant le jeu ayant couté le plus cher de l'histoire (300 millions de dollars) mais il faut prendre en compte que CIG developpe 2 jeux en même temps : Star Citizen, le MMO et Squadron 42, un jeu d'aventure se passant dans le même univers que Star Citizen.

Les joueurs évoluent dans notre univers 930 ans dans le futur. Le jeu est loin d'être fini. les joueurs peuvent pour le moment accéder à un seul système stellaire sur les dizaines prévus pour la realease, ce qui reste malgré tout immense et malgrés les bugs, les joueurs peuvent faire du combat spatial, des livraisons, du commerce, du minage...
